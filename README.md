# Portfolio — Aya Essqally

Portfolio personnel React (Vite + Tailwind + Framer Motion).

## Lancer le projet

```powershell
cd C:\Users\ayaes\Desktop\portfolio-aya
npm install
npm run dev
```

Ouvre l’URL affichée (généralement `http://localhost:5173`).

## Build production

```powershell
npm run build
npm run preview
```
